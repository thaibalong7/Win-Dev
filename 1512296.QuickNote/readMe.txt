- Các chức năng đã thực hiện được: 
	+ Lúc khởi động, chương trình được ẩn dưới vùng thông báo của hệ thống:
		* Exit: thoát khỏi chương trình;
		* View notes (DialogBox): tạo listview xem danh sách các tag đã lưu, khi nhấn vào mỗi tag thì 				 listview còn lại hiện danh sách các notes của tag đó. Khi nhấn vào notes thì một messageBox hiện 			 lên chứa đầy đủ các ký tự của notes;
		* View statistics (SubClass): hiển thị biểu dồ hình trụ các tag là số lượng mỗi tag.
	+ Double click vào icon dưới phần thông báo hoặc sử dụng tổ hợp Windows + Space để hiển thị cửa sổ Add 	 notes (thêm notes), nhấn SAVE để lưu lại sự thay đổi.
- Hướng dẫn sử dụng:
	+ Khởi động chương trình,  chương trình sẽ chạy ẩn và một icon của chương trình xuất hiện ở phần thông 	 	 báo. Chuột phải vào để xem menu các chức năng. View notes để xem các tag và các note đã lưu. View 	 	 statistics để xem biểu đồ thống kê tỉ lệ số lượng note của tag. Exit để thoát chương trình.
	+ Để thêm notes hay tag, Double click vào icon của chương trình dưới phần thông báo hoặc tổ hợp phím 	 Windows + Space. Nhập vào các textbox tương ứng và nhấn SAVE để lưu lại, khi nhấn lưu, các textbox sẽ 	 trở thành trống. 
- Các ghi chú khác: 
	+ File dữ liệu được ghi ở dạng utf8 nên có thể xem được bằng ứng dụng Notepad hoặc ứng dụng tương tự;
	+ File root là file chứa danh sách các tag đã được lưu, các file còn lại mang tên các tag chưa nội dung 	 các notes đã được lưu.
