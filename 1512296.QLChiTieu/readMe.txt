	+ Khi chương trình chạy lên tự động nạp dữ liệu trong file .txt lên và hiển thị ở dạng ListView.
	+ Khi thoát chương trình sẽ tự động lưu lại các mục chi vào file .txt.
	+ Vẽ biểu đồ thể hiện tỉ lệ tiêu xài.

	+ Lưu và nạp dữ liệu để hiển thị ở dạng tiếng Việt.
